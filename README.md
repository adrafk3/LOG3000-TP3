# LOG3000-TP3
# Calculatrice Web Flask

## Numéro d’équipe

**Équipe 36**

## Objectif

Ce projet consiste à mettre en place, documenter et améliorer une application web de calculatrice simple développée avec **Flask (Python)**.
L’objectif initial de cette étape est de configurer un dépôt GitHub collaboratif pour permettre le travail d’équipe, le suivi des modifications et la préparation des prochaines phases (documentation, tests, correction de bogues, etc.).

## Prérequis d’installation

Avant de commencer, assurez-vous d’avoir installé :

* **Git**
* **Python 3.x**
* **pip** (gestionnaire de paquets Python)
* Un **compte GitHub**

## Instructions d’installation

1. **Cloner le dépôt**

   * Remplacez `<url-du-repo>` par l’URL de votre dépôt GitHub :

   ```bash
   git clone <url-du-repo>
   ```

2. **Accéder au dossier du projet**

   ```bash
   cd <nom-du-projet>
   ```

3. **(Optionnel mais recommandé) Créer un environnement virtuel**

   ```bash
   python -m venv venv
   ```

4. **Activer l’environnement virtuel**

   * **Windows (PowerShell)** :

   ```bash
   .\venv\Scripts\Activate.ps1
   ```

   * **macOS / Linux** :

   ```bash
   source venv/bin/activate
   ```

5. **Installer les dépendances**

   * Si un fichier `requirements.txt` est fourni :

   ```bash
   pip install -r requirements.txt
   ```

   * Sinon, installer Flask manuellement :

   ```bash
   pip install flask
   ```
